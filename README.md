Dependency-less asar archive implementation

## Install via [npm](https://npmjs.com)

```sh
$ npm install --save rasa
```

## Usage

```js
var rasa = require( 'rasa' )
```

```js
var archive = new rasa.Archive()
```

## Benchmarks

Run the benchmarks

```
npm run benchmark
```
